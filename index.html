O que são Clients?

Um client é o dispositivo ou software que solicita recursos de um server. Exemplos: navegadores, apps de email ou dispositivos móveis. HTTP é o protocolo mais usado para comunicação entre clients e servers.

O que são Servers?

Um server é um computador que fornece serviços ou recursos a outros dispositivos (os clients). Exemplo: servidor web, servidor de banco de dados. Alguns servidores usam protocolos seguros como HTTPS para comunicação.

Como é uma estrutura básica de um HTML?

A estrutura mínima de um documento HTML inclui: tags desnecessárias, mas essenciais são:

  1. <!DOCTYPE html>
  2. <html>
  3. <head> com meta e título
  4. <body> com o conteúdo visível

Essa estrutura garante que o navegador interprete corretamente o conteúdo.

Entendendo a diferença entre listas ordenadas e não ordenadas

Listas ordenadas (<ol>) apresentam sequência numérica. Listas não ordenadas (<ul>) usam marcadores.

  • Vantagens do ul: fácil visualização de tópicos gerais
  • Vantagens do ol: ideal para instruções passo a passo
